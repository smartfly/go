# 时间工具转化

为了便于后期开发，整理开发相关常用时间工具，目前时间工具支持功能：
- time to time string
- timestamp to time string
- time string to time
- time string to timestamp

# Time

> time 零值
```
The zero value of type Time is January 1, year 1, 00:00:00.000000000 UTC.
```
